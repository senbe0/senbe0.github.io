    視聴者数をカウントするwebシステムを運用しています.
    I run a web system that counts the number of viewers.

    1.1 サイトの仕組み. How the site works.

    自作サーバで運用しています。 ドメインを取得してからネームサーバーへの登録、 リバースプロキシの設定まで、すべて自分で行いました。 コンテナ技術や、データベースのレプリケーションで負荷分散を行っています。
    pub/sub型のアーキテクチャを採用。ポーリングなどで余計な通信を行わない用に効率化した。
    PubSubHubbub Core 0.4プロトコルにしたがって、subscriber側のサーバーをfastAPIで実装した。

    マルチプロセスでインスタンスを管理しています。
